# yangtzeu-skill

> 长江大学技能交换网。

长江大学技能交换网(开发中)。使用React + Antd + Router + Sass + css-modules ，亦可作为脚手架使用。

## 使用指南

### 使用示例

```
$ npm install
$ npm start
```

or:

```
$ yarn
$ yarn start
```
